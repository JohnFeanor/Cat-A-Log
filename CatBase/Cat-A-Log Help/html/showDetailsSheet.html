The Show Details Window

When Cat-A-Log opens, you are presented with a window like this.

The left hand side gives details of the shows, and the right hand side gives details of the cats entered into that show.

To add a show, left click the '+' button

A show named 'New show' appears

To name the show, type the name of the show in the 'Name:' box,
then type (tab key)

Set the date of the show by typing the day (1 - 31),
type (tab key),
typing the month (1 - 12),
type (tab key),
then typing the year (e.g. 2012),
type (tab key).

Use the menu to choose whether the show is to be
QFA style (Longhair, Shorthair and Companion)
or ACF style (Group 1, Group 2, Group 3, and Group 4)

Type the number of rings in the show (between 1 to 6)
