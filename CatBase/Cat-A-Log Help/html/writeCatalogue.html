Writing a catalogue


When you have entered all the cats into your show, it is time to create the files necessary for the catalogue and challenge certificates. Select Write...' from the file menu.

This will create the following files

  • a catalogue.html file
  • a judges notes.html file
  • a challenges.xml file
  • a Catalog_files folder

The .html files can be opened in MSWord to give you the basis for your catalogue and judges notes. The cats will all be sorted into show order.

The .xml file can be opened in MS Exel and be used as a mail merge file to create your challenge certificates.
