Exporting and importing cats


To make a file of all the cats in your database to allow someone else to use on their computer, select 'Export cats...' from the file menu.

This will enable you to save a .cats file that lists all the cats in your database.
You may import the cats from another computer's database if they give you a .cats file. Select 'Import cats...' from the file menu. Find the .cats file and click Open.
