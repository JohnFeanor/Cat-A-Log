Adding entries to the show

After creating a show, you can add entries.

Click the button or type ()  to start entering a new cat or kitten.

This will bring up the entry sheet.
Type in the cat's registration number, or 'Pending' if a kitten or companion without a registration number yet.

As you type, the computer might predict what you mean.
If it predicts correctly, type (right arrow key) to accept what it has predicted.

Type (tab key) to start entering the next field.
If you type a name or registration number of a cat or kitten in the database, it will fill in the rest of the fields with that cat's information.
If the cat is not yet in the database, continue adding information into the appropriate box.
If the computer correctly predictes what you want to type in, type (right arrow key) to accept what it has predicted.
Type (tab key) to enter that box, and move to the next.

If the exhibitor is hiring a cage for this entry, select the hire button. If the entry is not in a standard (small) size cage or hiring a cage,
select the size cage that is nominated on the entry form - standard large size or other size.
If other size, type the measurement of the cage (in millimetres) into the box next to it.

Click on any rings the cat is not entered into to deselect them,
Tick the will work box if the exhibitor is willing to help at the show.

Click or type (enter) to accept the new entry,

Click or type (esc) to cancel the entry.

The cat is now entered into the show.
