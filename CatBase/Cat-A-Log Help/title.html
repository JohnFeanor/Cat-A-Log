The Cat Show catalogue generator

Cat-A-Log is a data base application to help sort and give cage numbers to cats entered into a cat show.

To use this application you:-

  • set the name and date of the show
  • enter a cat  into the data base.
  • edit a cat  already entered in the show.
  • export the cats  to a html file for inclusion into your catalogue. The cats are automatically sorted into show order.
